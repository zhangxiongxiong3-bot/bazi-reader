## ✨ 能做什么

把排盘 App（测测、万年历等）截图发给 Claude，自动识别并完成：

| 功能 | 触发说法 |
|------|---------|
| 🏠 六维综合分析 | "帮我全面看看命盘" |
| 🎭 性格画像 + 外形推测 | "描述我的性格" / "你猜猜我经历过什么" |
| ⚖️ 用神忌神分析 | "我的用神是什么" |
| 🏆 格局评级 | "我这个命格好不好" / "成格吗" |
| 📊 大运走势打分排名 | "0-70岁打分" / "哪段运气最好" |
| 💡 天赋挖掘 + 职业方向 | "我适合什么工作" |
| 📅 流年分析 | "2026年怎么样" |
| 🔟 十年逐年分析 | "未来十年" / "2025-2035年" |
| 🌱 运势提升建议 | "日常怎么提升运势" |
| 🎯 特殊情境 | 不婚 / 丁克 / 重新评分 |

## 🚀 安装使用

### 方法一：直接安装 .skill 文件（推荐）

1. 下载 [`bazi-reader.skill`](./bazi-reader.skill)
2. 在 Claude Cowork 桌面版中，进入 **Settings → Capabilities → Skills**
3. 点击 **Install from file** 选择下载的 `.skill` 文件
4. 安装完成，上传排盘截图即可触发

### 方法二：手动添加 SKILL.md

1. 复制 [`SKILL.md`](./SKILL.md) 的内容
2. 在 Claude Cowork 中创建新 Skill，粘贴内容保存

## 🎯 设计理念

传统命理的最大问题是**太晦涩**——一堆术语堆出来，普通人完全不知道对自己意味着什么。

这个 Skill 的目标：**把命理翻译成生活语言**。

- ❌ "日主辛金，印星偏弱，食伤当令，宜用印制食"
- ✅ "你就像一块还没完全锻造好的金属，现在最需要的是找一个稳定的环境打磨自己，不适合冲太猛"

每个分析都落地到**"你该怎么做"**，而不只是"命里有什么"。
